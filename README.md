### Prerequisites
Make sure you have:
- Node.js
- npm or yarn

### Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/TheFatPanda-Dev/weather-app-widget
cd weather-app-widget
npm install
# or
yarn install
```

## Run locally

```bash
npm start
```

Then open your browser and visit: http://localhost:3002
